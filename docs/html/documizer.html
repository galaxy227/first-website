GSDL.h void init()

Initialize gleeg.

Notes

Be sure to call end() to free memory allocated by init().

GSDL.h void end()

Clean memory associated with gleeg.

GSDL.h bool is_quit()

Check if the user has prompted to quit the application.

Return

Returns true once prompted to quit, such as clicking to close the application's window.

GSDL.h void update()

Update the game engine state, such as processing events.

GSDL.h void render()

Render the current game engine state.

GSDL.h float get_delta_time()

Check if the user has prompted to quit the application.

Return

Returns the time since last frame in seconds.

GSDL_log.h void log_debug_(const string& debug_message, const string& caller_file_name, const unsigned int caller_line)

Write a debug message to the log file located in the binary directory.

Parameters

debug_message: The message to be logged.

Notes

Do not provide more than one argument. The parameters caller_file_name and caller_line are managed by the preprocessor.

GSDL_log.h void log_error_(const string& error_message, const string& caller_file_name, const unsigned int caller_line)

Write an error message to the log file located in the binary directory.

Parameters

error_message: The message to be logged.

Notes

Do not provide more than one argument. The parameters caller_file_name and caller_line are managed by the preprocessor.

GSDL_file.h string get_exe_dir()

Get the directory path where the game's binary is stored.

Return

Path of specified directory returned as string with forward slash as trailing character.

GSDL_file.h string get_asset_dir()

Get the directory path where the game's data is stored.

Return

Path of specified directory returned as string with forward slash as trailing character.
